# Quiz app using spring boot

## Quiz app that can:

### 1) Upload question into a dataBase.
### 2) Get questions stored in the dataBase.
### 3) Create quiz based on difficulty and number of questions.
### 4) On submission calculates score.

## Dependencies used:

### 1) JPA + hibernate.
### 2) Spring Web.
### 3) PostgreSql Driver
### 4) JDBC
